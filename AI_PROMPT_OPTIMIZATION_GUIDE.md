# AI保险顾问提示词优化指南

## 优化日期
2026-01-01

## 优化概述
对 `api/ai_consultant_service.py` 中的AI提示词进行了全面优化，提升AI输出的专业性、结构化程度和实用价值。

---

## 🎯 优化目标

1. **专业性提升**：明确AI的专业资质和角色定位
2. **结构化输出**：确保AI按照固定格式输出高质量内容
3. **量化分析**：要求AI提供具体的数字和计算方法
4. **实用建议**：提供可操作的保险规划建议
5. **风险意识**：强调风险提示和注意事项
6. **客户导向**：避免过度推销，以客户利益为先

---

## 📋 优化内容详解

### 1. **角色定位优化**（System Prompt）

#### 优化前：
```
你是一位资深的保险规划顾问，拥有20年的从业经验。
```

#### 优化后：
```markdown
# 角色定位
你是一位专业的保险规划顾问，拥有以下资质：
- CFP国际认证理财规划师
- 20年以上保险行业从业经验
- 擅长家庭风险管理和财务规划
- 精通台湾保险市场和监管政策
```

**优化效果**：
- ✅ 明确专业资质（CFP认证）
- ✅ 强调领域专长（家庭风险管理）
- ✅ 突出市场熟悉度（台湾市场）
- ✅ 提升AI回答的权威性

# 核心原则

### 3. **分析任务结构化**

#### 优化前：
简单要求"分析客户需求并提供咨询报告"

#### 优化后：
分为**6个明确步骤**，每个步骤都有详细的分析要点：

```markdown
# 分析任务

## 第一步：深度需求分析
1. **人生阶段特征**：分析该阶段的典型风险和保险需求
2. **家庭责任评估**：评估家庭经济支柱的保障责任
3. **财务健康度**：分析收入、预算、负债的合理性
4. **保险需求优先级**：确定最紧迫的保障需求（1-5级）

## 第二步：风险评估
1. **身故风险**：如果客户身故，家庭会面临多大的财务缺口？
2. **重疾风险**：大病治疗费用和收入中断损失如何弥补？
3. **医疗费用风险**：日常医疗和住院费用的负担能力？
4. **长期照护风险**：失能或老年照护的资金准备？
5. **子女教育风险**（如适用）：教育资金是否充足？

## 第三步：保障缺口分析
针对每个风险点，计算：
- 现有保障额度（如有）
- 建议保障额度（基于科学方法）
- 保障缺口 = 建议额度 - 现有额度

**建议保障额度计算方法**：
- 寿险：年收入 × 10 + 未还贷款 + 子女教育金 - 现有流动资产
- 重疾险：年收入 × 3-5（治疗费+康复费+收入损失）
- 医疗险：根据医疗通膨和家庭病史评估
- 意外险：年收入 × 5-10

## 第四步：产品推荐
从提供的产品列表中，推荐最合适的3个产品，要求：
1. **必须选择列表中的产品**（使用product_id）
2. **匹配度优先**：优先考虑匹配度高的产品
3. **保障互补**：推荐的产品应覆盖不同风险
4. **预算合理**：总保费应在客户预算范围内（允许超出10-20%，但需说明理由）
5. **详细理由**：每个产品至少200字推荐理由

## 第五步：专业建议
1. **保障规划建议**：先保障后储蓄，先大人后小孩
2. **缴费期限建议**：根据年龄和收入稳定性建议缴费年期
3. **保额配置建议**：各类险种的保额如何搭配
4. **产品组合策略**：定期+终身、主险+附加险的组合思路
5. **未来调整方向**：3-5年后如何调整保单

## 第六步：风险提示
至少提供5条注意事项，包括但不限于：
- 健康告知的重要性
- 等待期的影响
- 保费缴纳能力评估
- 保单条款理解
- 受益人指定建议
```

**优化效果**：
- ✅ 分析流程清晰，逻辑严密
- ✅ 每个步骤都有明确的输出要求
- ✅ 提供科学的保额计算方法
- ✅ 强制要求详细的推荐理由（200字+）
- ✅ 确保风险提示的完整性

### 4. **JSON输出格式优化**

#### 优化要点：

##### A. 客户分析（customer_analysis）
**优化前**：简单的文字描述
**优化后**：
```json
"customer_analysis": "【客户需求深度分析】300-400字，包含：
1. 人生阶段特点及典型风险
2. 家庭责任量化分析
3. 财务状况健康度评估
4. 保险需求优先级排序（用1-5星标注）

示例：
该客户处于[人生阶段]，是家庭经济的[主要/次要]支柱..."
```

##### B. 风险评估（risk_assessment）
**优化后**要求：
```json
"risk_assessment": "【风险评估报告】200-300字，包含：
1. 最大风险点识别（如：身故、重疾、医疗等）
2. 每个风险的潜在财务损失（量化）
3. 风险紧急程度评级（高/中/低）
4. 现有保障的不足之处

示例：
若客户不幸身故，家庭将面临年收入XX万中断，房贷XX万需偿还..."
```

**关键改进**：要求量化财务损失，不再是空洞的描述

##### C. 保障缺口分析（coverage_gap）
**优化后**：
```json
"coverage_gap": {
    "寿险": "缺口分析：建议保额XX万（计算方式：年收入×10+房贷+教育金），现有XX万，缺口XX万",
    "重疾": "缺口分析：建议保额XX万（年收入×5，涵盖3-5年康复期），现有XX万，缺口XX万",
    "医疗": "缺口分析：建议每年XX万额度（覆盖住院、手术、特殊疾病），现有XX万，缺口XX万",
    "意外": "缺口分析：建议保额XX万（年收入×5-10），现有XX万，缺口XX万"
}
```

**关键改进**：
- ✅ 必须提供建议保额的计算方式
- ✅ 必须标注现有保障和缺口
- ✅ 所有数字都要量化

##### D. 推荐产品（recommended_products）
**优化前**：推荐理由简短（50-100字）
**优化后**：要求200-250字，分4个部分：

```json
"reason": "【详细推荐理由】200-250字：

1. 为什么选择这个产品？
分析客户的具体情况（年龄XX、收入XX、家庭状况XX），这个产品的[保障类型]正好契合客户的[具体需求]...

2. 它解决什么问题？
这个产品可以提供XX万的保障额度，弥补客户在[风险点]的缺口，如果发生[风险情况]，可以...

3. 优势在哪里？
相比其他产品，这个产品的优势在于：[特点1：如多次赔付]、[特点2：如保费豁免]、[特点3：如保障全面]...

4. 保费合理性？
年缴XX万，占年收入的XX%，符合保险规划的合理区间（10-15%），且在客户预算XX万范围内..."
```

**关键改进**：
- ✅ 推荐理由具体化、结构化
- ✅ 必须分析保费合理性
- ✅ 必须说明产品优势
- ✅ 必须解释如何解决客户问题

##### E. 专业建议（professional_advice）
**优化后**：要求400-500字，分5个部分：

```markdown
## 一、保障规划建议
根据客户情况，建议采用'先保障后储蓄'的策略...

## 二、缴费期限建议
考虑到客户XX岁，职业XX，收入稳定性XX，建议选择XX年缴费期...

## 三、保额配置建议
- 寿险：XX万（年收入×10的原则）
- 重疾险：XX万（年收入×3-5）
- 医疗险：XX万/年（覆盖住院+门诊）
- 意外险：XX万（年收入×5-10）

## 四、产品组合策略
建议采用'定期+终身'的组合...

## 五、未来调整方向
- 3年后：当子女进入XX阶段...
- 5年后：当家庭责任减轻...
- 10年后：当接近退休期...
```

**关键改进**：
- ✅ 结构化的建议框架
- ✅ 提供具体的保额数字
- ✅ 包含长期规划（3/5/10年）

##### F. 预算规划（budget_planning）
**优化后**：
```json
"budget_planning": {
    "recommended_total": 105000,
    "breakdown": {
        "寿险": 40000,
        "重疾险": 35000,
        "医疗险": 20000,
        "其他": 10000
    },
    "budget_ratio": "17.5%（年收入60万的17.5%，略高于标准15%，但考虑到家庭责任重大，属于合理范围）",
    "payment_strategy": "建议采用月缴方式，每月约8,750元，减轻一次性缴费压力。如果资金充裕，可选择年缴享受保费折扣（约95折）。"
}
```

**关键改进**：
- ✅ 详细的保费分解
- ✅ 预算占收入比的合理性说明
- ✅ 具体的缴费策略建议

##### G. 注意事项（warnings）
**优化前**：简单列举
**优化后**：每条都要详细说明（80-100字）

示例：
```
"⚠️ 健康告知的重要性：投保时必须如实告知健康状况，包括既往病史、家族病史等。若未如实告知，理赔时可能被拒赔。建议在填写健康告知前，仔细阅读条款或咨询专业人员。"

"⚠️ 保费缴纳能力评估：年缴保费XX万，占年收入XX%。请确保未来XX年内有稳定收入支持保费缴纳。若中途断缴，可能面临保单失效风险。建议预留6个月保费作为紧急备用金。"
```

**关键改进**：
- ✅ 每条注意事项都要说明后果
- ✅ 提供具体的建议
- ✅ 使用⚠️图标增强视觉效果

##### H. 保障计划（protection_plan）
**优化后**：分为短期、中期、长期三个阶段

```json
"protection_plan": {
    "immediate_protection": "【立即需要的保障】（0-1年）
优先配置推荐的3个产品，快速建立基础保障网。总保费约XX万，提供XX万保障额度，覆盖身故、重疾、医疗三大风险。这是最紧迫的保障需求，建议1个月内完成投保。",

    "medium_term_plan": "【中期保障规划】（3-5年）
随着收入增长和家庭变化，建议：1）增加重疾险保额至XX万；2）为子女配置教育金保险；3）考虑增加意外险保额；4）评估是否需要长期照护保险。预计每年增加保费XX万。",

    "long_term_plan": "【长期保障规划】（5年以上）
当子女成年、房贷还清后，家庭责任逐渐减轻，保障重点转向：1）退休规划；2）医疗升级；3）长期照护；4）资产传承。"
}
```

**关键改进**：
- ✅ 提供完整的时间线规划
- ✅ 每个阶段都有具体建议
- ✅ 帮助客户理解保险的动态调整
