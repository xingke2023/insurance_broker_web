## 数据模型

### InsuranceProduct 字段说明

#### 基本信息
- **company**: 所属保险公司（外键 → InsuranceCompany）
- **product_name**: 产品名称（例如：环宇盈活、世誉财富）
- **description**: 产品描述
- **payment_period**: 缴费年期（年数，如：1年、2年、5年）
- **annual_premium**: 年缴金额（港币）

#### 数据表格
- **surrender_value_table**: 退保发还金额表（JSON格式）
  ```json
  [
    {
      "year": 1,
      "guaranteed": 0,
      "non_guaranteed": 0,
      "total": 0
    },
    {
      "year": 2,
      "guaranteed": 2500,
      "non_guaranteed": 19200,
      "total": 21700
    }
  ]
  ```

- **death_benefit_table**: 身故保险赔偿表（JSON格式）
  ```json
  [
    {
      "year": 1,
      "benefit": 100000
    },
    {
      "year": 2,
      "benefit": 150000
    }
  ]
  ```

#### AI推荐元数据（待完善）
- **target_age_min**: 目标年龄段最小值（如：25）
- **target_age_max**: 目标年龄段最大值（如：50）
- **target_life_stage**: 目标人生阶段（多个用逗号分隔）
  - 扶幼保障期
  - 收入成长期
  - 责任高峰期
  - 责任递减期
  - 退休期

- **coverage_type**: 保障类型（多个用逗号分隔）
  - 储蓄
  - 重疾
  - 医疗
  - 教育基金
  - 退休规划

- **min_annual_income**: 最低年收入要求（港币）
- **features**: 产品特点列表（JSON数组）
  ```json
  [
    "高额身故保障",
    "现金价值稳定增长",
    "可附加重疾保障"
  ]
  ```

- **ai_recommendation_prompt**: AI推荐提示词（帮助AI理解产品特性）

#### 产品特性
- **is_withdrawal**: 是否包含提取功能（布尔值）
- **is_active**: 是否启用（布尔值）
- **sort_order**: 排序（数字）

#### 时间戳
- **created_at**: 创建时间
- **updated_at**: 更新时间

## 数据维护

### 添加新产品
1. 进入Django Admin后台
2. 导航到 "保险公司产品" 模块
3. 点击 "添加保险公司产品"
4. 填写基本信息和表格数据
5. 保存

### 批量导入
可以使用 Django Management Command 批量导入产品数据：
```bash
python manage.py import_products data/products.json
```

### 数据导出
```bash
python manage.py dumpdata api.InsuranceProduct --indent 2 > products_backup.json
```
