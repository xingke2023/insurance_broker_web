# 表格提取规则说明

## 核心规则：严格过滤"保单年度终结"列

### 规则概述

只处理**在表头（第一行）包含"保单年度终结"或"保單年度終結"列**的表格，其他表格一律跳过。

---

## 两层过滤机制

### 第1层：AI分析阶段（tablesummary）

**位置**：`extract_table_summary()` 函数

**提示词要求**：
```
**核心要求（必须严格遵守）：**
1. 必须包含"保单年度终结"或"保單年度終結"列
2. 检查列标题：必须在表格的第一行（表头）找到这个列名
3. 只输出包含"保单年度终结"列的表格，其他表格一律不输出！
```

**输出结果**：`文件_tablesummary.txt`
- ✅ 只包含有"保单年度终结"列的表格
- ❌ 不包含其他类型的表格

---

### 第2层：HTML提取阶段（正则匹配）

**位置**：`extract_tables_with_year_column()` 函数

**检查逻辑**：
```python
# 1. 提取表格的第一行（表头）
first_tr_pattern = re.compile(r'<tr[^>]*>(.*?)</tr>', re.DOTALL | re.IGNORECASE)
first_tr_match = first_tr_pattern.search(table_html)

# 2. 检查表头是否包含"保单年度终结"列
year_keywords = ['保单年度终结', '保單年度終結']
has_year_column = any(keyword in first_row for keyword in year_keywords)

# 3. 只保留包含该列的表格
if has_year_column:
    tables_with_year.append(table_tag)
```

**输出结果**：`文件_table1_html.txt`, `文件_table2_html.txt` ...
- ✅ 每个文件都是包含"保单年度终结"列的合并表格
- ❌ 不包含其他类型的表格HTML

## 关键字匹配

支持以下两种写法（繁简体）：
- `保单年度终结` （简体）
- `保單年度終結` （繁体）

**注意**：必须完整匹配，部分匹配无效：
- ✅ `保单年度终结` → 匹配
- ✅ `保單年度終結` → 匹配
- ❌ `保单年度` → 不匹配
- ❌ `年度终结` → 不匹配
- ❌ `保单年度结束` → 不匹配

## 常见问题

### Q1: 为什么有些表格被跳过了？
**A**: 检查表格的表头（第一行）是否包含"保单年度终结"或"保單年度終結"列名。只有包含这个列的表格才会被处理。

### Q2: 如果表格中间某一列叫"保单年度终结"，但不在表头？
**A**: 不会被处理。必须在**表头（第一行）**找到这个列名。

### Q3: 表格标题中包含"保单年度"，会被处理吗？
**A**: 不会。必须在**表格的列标题（<th>或<td>）**中找到完整的"保单年度终结"字样。

### Q4: 如果需要处理其他类型的表格怎么办？
**A**: 修改 `year_keywords` 列表，添加新的关键词：
```python
year_keywords = [
    '保单年度终结',
    '保單年度終結',
    '你的新关键词'
]
```
