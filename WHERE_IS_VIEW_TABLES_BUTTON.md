## 🔧 如果按钮没有显示

### 检查清单

1. **✅ 确认文档已完成处理**
   - 状态应该是"已完成"
   - 不是"处理中"或"失败"

2. **✅ 确认文档有表格数据**
   - 在Admin后台查看：http://localhost:8007/admin/api/plantable/
   - 搜索对应的文档ID

3. **✅ 刷新页面**
   - 按 Ctrl+F5 强制刷新
   - 清除浏览器缓存

4. **✅ 检查浏览器控制台**
   - 按 F12 打开开发者工具
   - 查看 Console 是否有错误
   - 查看 Network 请求是否成功

### 常见问题

**Q: 卡片显示了但是没有表格？**
```
A: 可能是：
   - 文档处理尚未完成
   - OCR内容中没有"保单年度终结"相关表格
   - 需要等待Celery任务完成
```

**Q: 点击按钮没有反应？**
```
A: 可能是：
   - 浏览器阻止了弹出窗口
   - JavaScript错误，查看控制台
   - 网络问题，检查API请求
```

**Q: 表格显示为空？**
```
A: 可能是：
   - HTML源代码提取失败
   - 数据库中html_source字段为空
   - 在Admin后台检查原始数据
```

## 🎓 进阶功能

### 1. 下载HTML文件
在表格详情页，点击右上角"下载HTML"按钮，可以：

在表格详情页，点击"查看HTML源代码"：
- 查看完整的HTML标签
